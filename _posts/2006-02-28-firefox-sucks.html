Yes, seriously:

  • It crashes when I try to close a panel with flash (not every flash, but there are some that crash it regularly), also crashes sometimes when I right click flash (don't ask me why I do that, I just have to click everywhere all the time).
  • It gradually eats up all the memory. If I fire it up in the morning, by the evening all my 512mb of ram is just gone and I have to close firefox (if I don't, it fills up the memory completely and crashes (sometimes even crashes the X server)).
  • It takes ages to render some pages, especially some javascript can cause Firefox to go insane, using 100% of processor time for several (dozens) seconds. Scrolling on such a page is almost impossible.

Well, that's about it. So I tried Opera now that it is free and all, but there is the problem that it takes ages to actually download a page (while rendering is nice and fast (for example from localhost (even with hundreds of pictures))). I am not sure why is that, I have disabled ipv6 and know no other reason that could slow it down.

And I am not really used to Opera's UI, so I'd prefer to somehow fix firefox (btw it is 1.0.7, will not install anything newer for now because my distribution doesn't support it and I don't want to end in dependency hell).
